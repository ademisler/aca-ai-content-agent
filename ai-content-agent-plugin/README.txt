AI-powered content creation and management plugin that generates blog posts, ideas, and manages your content workflow automatically.

== Description ==

AI Content Agent (ACA) is a powerful WordPress plugin that revolutionizes your content creation process using artificial intelligence. Create high-quality blog posts, generate content ideas, and manage your entire content workflow with minimal effort.

= Key Features =

* **AI-Powered Content Generation**: Create complete blog posts with title, content, meta descriptions, and focus keywords
* **Smart Idea Generation**: Generate unlimited content ideas based on your writing style
* **Style Guide Analysis**: AI analyzes your existing content to maintain consistent tone and style
* **Multiple Automation Modes**: Choose from Manual, Semi-Automatic, or Full-Automatic content creation
* **Image Integration**: Generate AI images or fetch from stock photo services (Pexels, Unsplash, Pixabay)
* **SEO Optimization**: Built-in SEO features with meta titles, descriptions, and focus keywords
* **Content Calendar**: Visual calendar to manage and schedule your content
* **Activity Logging**: Track all content creation activities
* **Draft Management**: Create, edit, and publish drafts seamlessly

= Automation Modes =

* **Manual Mode**: Full control over when to generate ideas and create content
* **Semi-Automatic Mode**: Automatic idea generation with manual content creation
* **Full-Automatic Mode**: Complete automation from idea generation to publishing

= AI Integration =

The plugin uses Google's Gemini AI for:
- Content style analysis
- Blog post generation
- Idea creation
- Image generation

= Stock Photo Integration =

Support for major stock photo services:
- Pexels
- Unsplash
- Pixabay

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ai-content-agent` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to AI Content Agent in your WordPress admin menu
4. Add your Google AI API key in the Settings tab
5. Generate your first Style Guide by clicking "Analyze Style Guide"
6. Start creating content!

== Frequently Asked Questions ==

= Do I need a Google AI API key? =

Yes, you need a Google AI API key to use the content generation features. You can get one from the Google AI Studio.

= Can I use my own images? =

Yes, you can either generate AI images or connect to stock photo services like Pexels, Unsplash, or Pixabay.

= Will this work with my SEO plugin? =

The plugin is compatible with popular SEO plugins like RankMath and Yoast SEO.

= Can I edit the generated content? =

Absolutely! All generated content can be fully edited before publishing.
