== Installation ==

1. Upload the plugin files to `/wp-content/plugins/ai-content-agent/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure your Gemini API key in AI Content Agent â†’ Settings
4. For Pro features, purchase and activate your license key

== Frequently Asked Questions ==

= How do I get Pro features? =

Purchase a Pro license from our Gumroad store, then activate it in the plugin settings under "Pro License Activation".

= What API keys do I need? =

* **Required**: Google Gemini API key (for AI content generation)
* **Optional**: Image service API keys (Pexels, Unsplash, Pixabay)
* **Pro Only**: Google Search Console integration

= Is my data secure? =

Yes. All license verification uses secure HTTPS connections, and your data is stored safely using WordPress standards.

= Can I upgrade from free to Pro? =

Absolutely! Purchase a Pro license and activate it in settings. All your existing content and settings will be preserved.
